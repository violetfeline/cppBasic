**Problem 9: Բալանսավորված փակագծեր**

Մուտքում տրվում է ‘(‘, ‘)’, ‘{‘, ‘}’, ‘[‘ և ‘]’ սիմվոներից բաղկացած հաջորդականություն, պահանջվում է արտածել YES, եթե այն փակագծերի ճիշտ արտահայտություն է և NO՝ հակառակ դեպքում։ (դաս 14)

**Examples**

Input
```
(){[]}[{{}}]
```
Output
```
YES
```
Input
```
((())
```
Output
```
NO
```
Input
```
[{]}
```
Output
```
NO
```
