**Problem 4: Պարզ թվեր**

Գրել ծրագիր, որը մուտքում կստանա դրական թիվ և կարտածի մինչև այդ թիվը եղած բոլոր պարզ թվերը։ Արտածվող թվերը միմյանցից պետք է բաժանված լինեն բացատանիշով։ (դաս 5, 6)

**Examples**

Input
```
2
```

Output
```
2
```

Input
```
5
```

Output
```
2 3 5
```

Input
```
15
```

Output
```
2 3 5 7 11 13
```
