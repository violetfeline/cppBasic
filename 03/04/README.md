**Problem 4: Ենթաբառ**

Գրել ծրագիր, որը մուտքում կստանա մեկ բառ (ոչ ավել քան 20 սիմվոլ պարունակող) և մեկ սիմվոլ։ Անհրաժեշտ է արտածել մուտքագրված բառում՝ տրված սիմվոլի առաջին հանդիպման դիրքից աջ ընկած ենթաբառը։ (դաս 9)
