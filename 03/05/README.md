**Problem 5: Ենթաբառ ֊ 2**
Գրել ծրագիր, որը մուտքում կստանա S1 բառը (ոչ ավել քան 20 սիմվոլ), S2 բառը (ոչ ավել քան S1 բառում սիմվոլների քանակը) և կարտածի ‘YES’, եթե S2 հանդիսանում է ենթաբառ S1֊ից և ‘NO’ հակառակ դեպքում: (դաս 9)

**Examples**

Input
```
abcdefgh
abc
```

Output
```
YES
Input
abcd!@#$1234
#$12
```

Output
```
YES
```

Input
```
abcdefg
ABC
```

Output
```
NO
```
