**Problem 2: Անագրամ**

Գրել ծրագիր, որը մուտքում կստանա S1 և S2 բառերը (ոչ ավել քան 20 սիմվոլ) և կարտածի ‘YES’, եթե մուտքագրված բառերը հանդիսանում են անագրամ և ‘NO’ հակառակ դեպքում: (դաս 9)

**Examples**

Input
```
alo
ola
```
Output
```
YES
```
Input
```
coronavirus
carnivorous
```
Output
```
YES
```
Input
```
ABCDE
ABCDe
```
Output
```
NO
```
